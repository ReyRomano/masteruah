## Commit inicial

- Añadir al README.md los comanddos utilizados hasta ahora
y hacer un coomit inicial con el mensaje **commit inicial**.

		git status
		git add .
		git status
		git commit -m "commit inicial"

## Push inicial

- Subir los cambios al repositorio remoto.

		git branch -M main
		git push -u origin main
